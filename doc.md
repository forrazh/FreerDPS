Mes travaux avant la réunion ont porté sur une réécriture de FreeSpec avec Mathcomp/Monae, je suis actuellement entrain de nettoyer et de publier le code sur Github (a priori, tous mes travaux de thèse sont open source). 
Ces derniers temps, j'ai principalement travaillé sur la soumission d'un papier à COMPAS (conférence francophone de système). Dans ce papier, je raconte, entre autre, que l'objectif est d'améliorer FreeSpec pour y ajouter le support des probabilités (ce qui de mes recherches n'a jamais été fait par la communauté système du point de vue preuve assistée).

Dans cette perspective, j’avais imaginé arriver avec quelques problématiques concrètes autour des systèmes distribués probabilistes, afin de bénéficier de ton expertise sur ces questions. Cela dit, je serais bien sûr très intéressé d’avoir ton point de vue sur les directions de recherche les plus pertinentes, et ouvert à travailler sur des sujets que tu jugerais particulièrement intéressants ou prometteurs dans ce cadre.
